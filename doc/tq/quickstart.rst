安装天勤软件并启动
--------------------------------------------------------------------------
天勤软件目前只支持 Windows 操作系统。要安装天勤软件，首先请到 `天勤官网 <https://www.shinnytech.com/tianqin>`_ 下载安装包

下载完毕后，双击安装包进行安装。安装完毕后，点击桌面图标 【天勤】 开始运行

选择期货公司，输入账号密码并登录。如果您没有期货公司账号，可以使用【快期模拟】登录。
