.. _tutorial:

入门教程
=================================================

通过运行示例，来学习如何使用 TqSdk。所有的示例都在 demo 的文件下。

文件位置
-------------------------------------------------

+ 通过 `pip` 安装，:ref:`如何找到文件安装位置?<faq-pip-install-files-location>`。
+ 通过 github 安装，示例文件在 `tqsdk-python/tqsdk/demo` 下。


如何运行
-------------------------------------------------
.. code-block:: bash
    :caption: 运行文件

    $ python demo/t10.py
