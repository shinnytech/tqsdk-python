安装
=================================================
安装前准备:

* windows或linux系统
* python3.6+
* 安装天勤终端 0.8 以上版本

从github下载 tqsdk::

    git clone https://github.com/shinnytech/tqsdk-python.git


安装::

    python setup.py install


运行:

    * 启动天勤终端, 并登录模拟交易

    * 运行demo目录下的任一程序::

        python demo/t10.py
